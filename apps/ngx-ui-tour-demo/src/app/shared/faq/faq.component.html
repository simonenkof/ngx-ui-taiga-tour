  2. Add CSS for the div
  3. Use previously defined tour anchor

To add some space between backdrop edges and active tour anchor element, you can use .touranchor--is-active CSS class. Use it to add the required amount of padding and then subtract the same amount using negative margin which will keep overall page layout unchanged.
